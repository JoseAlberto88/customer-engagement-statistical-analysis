## 🎯 Project Overview  
This repository contains the full statistical analysis completed for the DAMO‑500 take‑home assignment. The dataset includes customer‑level behavioral and demographic variables used to evaluate engagement, spending, satisfaction, and conversion outcomes.

The analysis follows the complete workflow taught in Modules 1–7:
- Data preparation & screening  
- Outlier detection (IQR & Z‑scores)  
- Descriptive statistics  
- Probability modeling  
- Correlation & Chi‑square association tests  
- ANOVA diagnostics  
- Kruskal–Wallis non‑parametric comparison  
- Python simulations
